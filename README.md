# 🛍️ End-to-End Retail Analytics Dashboard | SQL Server + Power BI

## 📌 Project Overview

This project is an **End-to-End Retail Analytics Dashboard** developed using **SQL Server** and **Power BI**. It provides actionable insights into retail sales, store performance, product performance, inventory management, and customer behavior through interactive dashboards.

The project demonstrates the complete Business Intelligence workflow, including data modeling, data transformation, DAX calculations, KPI creation, and dashboard development.

---

## 🚀 Tech Stack

- Microsoft SQL Server
- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel
- Star Schema Data Modeling

---

## 📂 Data Model

The dashboard is built using a **Star Schema** consisting of the following tables:

### Fact Tables
- Fact_Sales
- Fact_Inventory

### Dimension Tables
- Dim_Product
- Dim_Customer
- Dim_Store
- Dim_Calendar
